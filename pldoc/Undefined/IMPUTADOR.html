public T_STATUSIMPUS SETIMPUTATIONS(TDAYIMPUTATIONS IN IMPUTADOR.T_DAYIMPUTATIONS DEFAULT NULL, 
                                    STARTDATE IN DATE DEFAULT NULL, 
                                    ENDDATE IN DATE DEFAULT NULL, 
                                    TMODELDAYS IN IMPUTADOR.T_MODELDAYS DEFAULT NULL, 
                                    RANDOMTIME IN BOOLEAN DEFAULT FALSE, 
                                    RANDOMEFECTIVE IN BOOLEAN DEFAULT FALSE)
Set imputations.
Parameters:
TDAYIMPUTATIONS - Optional, if you imputation is for each day
STARTDATE - Optional, if you imputation is for model day. Start date
ENDDATE - Optional, if you imputation is for model day. End date
TMODELDAYS - Optional, if you imputation is for model day. 5 model days -> mondays-fridays
RANDOMTIME - Add random in start time, end time -> 5 minutes before or after
RANDOMEFECTIVE - Add random in efective time -> 15 minutes minus
Returns:
Type all set imputations done
